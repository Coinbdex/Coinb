Specifications:
==================



---


• 51% attacks resistant

• Adopts CSV technology

• Hybrid PoW/PoS

• Block time of 30 seconds

• Prefix: COINB adresses start with "C"  

• Uses a fork of Dash's DarkGravityWave for the difficulty algorithm
